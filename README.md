# html-css-barbeariaAlura

Vim compartilhar mais um projetinho que concluí quando estava iniciando HTML e CSS.

Neste projetinho eu abordei noções básicas de HTML e sua estrutura, noções básicas de CSS e como definir estilos e posicionamentos para elementos utilizando o CSS, diferenças entre os displays inline e block, pseudo-classes do CSS, formulários complexos, tabelas, inputs, transformações e transições, importação de conteúdo externo na página HTML, seletores avançados no CSS, viewport e design responsivo.
